# Chatbot

AI Chatbot of chatFoodie.

## Environment
> using python 3.11.2

### set python virtual environment
* Linux  
    ```sh
    $ activate_venv.sh
    (.venv) $
    ```
* Windows  
    ```cmd
    > activate_venv.bat
    (.venv) >
    ```

### install packages
```shell
(venv) $ pip install -r requirements.txt
```
